## Requirements
* Apache with rewrite module enabled
* PHP 7.4 or later
* MySql

## Features
* HTML Templates (Templates are made with just HTML, no need learn PHP and MySQL)
* Fully Blog System
* Advanced Admin Panel (Configure your blog as you like)
